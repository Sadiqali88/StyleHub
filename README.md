# 🚀 StyleHub

StyleHub is a full-stack multi-role e-commerce platform that connects Designers, Sellers, Buyers, and Printers in a single ecosystem.

It is designed to simplify the workflow of design-based product selling and dropshipping by allowing multiple roles to interact on one platform.

---

## ✨ Features

### 👤 User Roles
- **Designer** → Upload and manage designs  
- **Seller** → Create and list products using designs  
- **Buyer** → Browse products, add to cart, and place orders  
- **Printer** → View and process incoming orders  

---

## 🔄 Platform Workflow

Designer → Upload Design
↓
Seller → Create Product
↓
Buyer → Add to Cart & Place Order
↓
Printer → Receive & Process Orders
---

## 🛠 Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **API Handling:** Axios
